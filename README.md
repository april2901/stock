LOL-클라이언트 보조 프로그램
==============
### 친구의 인게임 여부확인/ 인게임시 상대 룬 확인
### 클라이언트 켜놓기 부담될때 사용
### api key 와 namelist, my_name는 사용자에 맞게 입력
### namelist는 친구의 롤닉네임 입력 ("" 안에)
